# Ask Me App

Ask Me is a simple Python-based interactive application that allows users to ask questions and receive responses through a structured and modular codebase. The project follows clean project structure, virtual environment isolation, and best practices for running Python applications.

##  Project Structure
ask-me-app/  
│  
├── src/  
│   ├── __init__.py   
│   ├── main.py        
│   ├── managers.py               
│   ├── models.py         
│   └── utils.py            
│
├── requirements.txt     
├── README.md  
└── .gitignore  

##  Requirements

- Python 3.8+
- Git
- Virtual environment support (venv)

##  Installation & Run Steps

```bash
# Clone the Repository
git clone https://github.com/your-username/AskMe.git
cd ask-me-app

# Create a Virtual Environment
python -m venv venv

# Activate the Virtual Environment
# On Windows
venv\Scripts\activate

# On macOS / Linux
source venv/bin/activate

# install requirments
pip install -r requirements.txt

# Run the project
python -m src.main

```
